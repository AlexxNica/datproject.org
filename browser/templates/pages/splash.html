Publish all bits.

We're a not for profit service dedicated to making it easy to share containers and data together

Step 0: Install

npm install -g dat

Step 1: Create

Add your entire code base, source data, and dependencies.

Step 2: Share

Whenever you're ready, `dat publish` and publicbits will create a public repository just for you.

Step 3: Iterate

At any time, you can `dat push` and publicbits will update your repository.
